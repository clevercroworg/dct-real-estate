# DCT Real Estate

A modern, responsive real estate website built for DCT Real Estate to showcase their portfolio of residential and commercial projects. This application replaces the legacy static site with a dynamic, performance-optimized Next.js solution.

## 🚀 Tech Stack

This project is built using the following technologies:

-   **Framework:** [Next.js](https://nextjs.org/) (App Router)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **Icons:** [Ionicons](https://ionic.io/ionicons)
-   **Deployment:** Vercel (Recommended)

## 🚀 Deployment Instructions (Vercel)

Since the Next.js app is located in the `web` folder, you must configure Vercel as follows:

1.  Connect your GitHub repository to Vercel.
2.  In the **Configure Project** step, find the **Root Directory** setting.
3.  Click **Edit** and select the `web` folder.
4.  Proceed with deployment.

If you have already deployed and are seeing a 404 error:
1.  Go to your Vercel Project Dashboard.
2.  Navigate to **Settings** > **General**.
3.  Update the **Root Directory** to `web`.
4.  Go to **Deployments** and redeploy the latest commit.

## 📂 Project Structure

-   `/web`: The main Next.js application source code.
-   `/images`: Static image assets used in the project.
-   `/videos`: Video assets.
-   `/old files`: Backup of the original legacy HTML/CSS site.

## 🛠️ Features

-   **Dynamic Routing:** Project listings and detail pages are dynamically generated.
-   **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
-   **Modern UI:** Implements a clean, premium design language with smooth transitions and interactions.
-   **Component-Based:** Modular architecture for easier maintenance and scalability.
